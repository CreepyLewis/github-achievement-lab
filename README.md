## 👋 What is this?

This repo is a step-by-step lab for earning GitHub achievements. Whether you're just starting out or going for platinum tiers — this guide has you covered!

## 🎯 All GitHub Achievements

| Achievement | Description | Difficulty | Max Tier |
|-------------|-------------|------------|----------|
| [⚡ Quickdraw](./achievements/quickdraw.md) | Close an issue/PR within 5 minutes of opening | 🟢 Easy | 1 tier |
| [🎉 YOLO](./achievements/yolo.md) | Merge a PR without a code review | 🟢 Easy | 1 tier |
| [❤️ Heart On Your Sleeve](./achievements/heart-on-your-sleeve.md) | React with ❤️ on GitHub | 🟢 Easy | 1 tier |
| [🤝 Pair Extraordinaire](./achievements/pair-extraordinaire.md) | Co-author a merged pull request | 🟡 Medium | 💎 Platinum |
| [🦈 Pull Shark](./achievements/pull-shark.md) | Get pull requests merged | 🟡 Medium | 💎 Platinum |
| [🌊 Open Sourcerer](./achievements/open-sourcerer.md) | Merge PRs in multiple public repos | 🟡 Medium | 1 tier |
| [🦊 Galaxy Brain](./achievements/galaxy-brain.md) | Get answers accepted in Discussions | 🔴 Hard | 💎 Platinum |
| [⭐ Starstruck](./achievements/starstruck.md) | Get stars on your repository | 🔴 Hard | 💎 Platinum |
| [🔒 Secure](./achievements/secure.md) | Get a security report accepted | 🔴 Very Hard | 1 tier |

## 🚀 Quick Start

New to GitHub achievements? Start with the easiest ones:

1. ⚡ [Quickdraw](./achievements/quickdraw.md) — takes 5 minutes
2. 🎉 [YOLO](./achievements/yolo.md) — takes 5 minutes
3. ❤️ [Heart On Your Sleeve](./achievements/heart-on-your-sleeve.md) — takes 1 minute
4. 🤝 [Pair Extraordinaire](./achievements/pair-extraordinaire.md) — takes 10 minutes

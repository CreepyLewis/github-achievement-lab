# 🤝 Pair Extraordinaire

> Co-author a merged pull request using the `Co-authored-by` trailer in a commit message.

**Difficulty:** 🟡 Medium  
**Tiers:** 4 (Bronze → Silver → Gold → Platinum)  
**Time to earn:** 10 minutes

---

## 🏆 Tiers

| Tier | Co-authored PRs Merged |
|------|----------------------|
| 🥉 Bronze (x2) | 2 |
| 🥈 Silver (x3) | 10 |
| 🥇 Gold (x4) | 24 |
| 💎 Platinum (x5) | 48 |

---

## 📋 How to Earn It

1. Go to any of your repositories
2. Edit any file (e.g. add a line to README)
3. In the **Title box** write your commit message e.g. `Update README`
4. In the **Description box** (below the title) add:
```
Co-authored-by: FRIEND_USERNAME <FRIEND_USERNAME@users.noreply.github.com>
```
5. Select **"Create a new branch"**
6. Click **"Propose changes"**
7. Click **"Create pull request"**
8. Click **"Merge pull request"**
9. Click **"Delete branch"** to keep your repo clean 🧹

---

## ✅ Tips

- Use your friend's **GitHub no-reply email** — format: `username@users.noreply.github.com`
- You can co-author with **multiple friends** in one commit:
```
Co-authored-by: friend1 <friend1@users.noreply.github.com>
Co-authored-by: friend2 <friend2@users.noreply.github.com>
```
- Your friend does **NOT** need to be a collaborator on the repo
- Your friend does **NOT** need to do anything — they get credited automatically!
- Using the **same friend every time** still counts toward tiers
- Every co-authored PR also counts toward **Pull Shark** 🦈

---

## ⚠️ Common Mistakes

- Putting the `Co-authored-by` line in the **title box** — it must be in the **description box**
- Using the wrong email format — always use `username@users.noreply.github.com`
- Committing directly to `main` — must use a **branch** and merge via PR

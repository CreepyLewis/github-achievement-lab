# 🎉 YOLO

> Merge a pull request without a code review.

**Difficulty:** 🟢 Easy  
**Tiers:** 1 (no tiers)  
**Time to earn:** 5 minutes

---

## 📋 How to Earn It

1. Go to any of your **public repositories**
2. Edit any file (e.g. add a line to README)
3. Select **"Create a new branch"** when committing
4. Click **"Propose changes"**
5. Click **"Create pull request"**
6. Click **"Merge pull request"** — without requesting any review!
7. Confirm merge ✅

---

## ✅ Tips

- Works on **your own repo** — no collaborators needed
- Make sure there are **no required reviewers** set up in branch protection rules
- The repo must be **public**

---

## ⚠️ Common Mistakes

- Having **branch protection rules** that require reviews — disable them first
- Merging into a repo that requires approvals

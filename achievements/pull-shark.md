# 🦈 Pull Shark

> Get pull requests merged.

**Difficulty:** 🟡 Medium  
**Tiers:** 4 (Bronze → Silver → Gold → Platinum)  
**Time to earn:** 5 minutes per PR

---

## 🏆 Tiers

| Tier | PRs Merged |
|------|-----------|
| 🥉 Bronze (x2) | 2 |
| 🥈 Silver (x3) | 16 |
| 🥇 Gold (x4) | 128 |
| 💎 Platinum (x5) | 1024 |

---

## 📋 How to Earn It

1. Go to any of your **public repositories**
2. Edit any file
3. Select **"Create a new branch"** when committing
4. Click **"Propose changes"**
5. Click **"Create pull request"**
6. Click **"Merge pull request"**
7. Delete the branch 🧹
8. **Repeat!**

---

## ✅ Tips

- Every PR you merge counts — even tiny edits like fixing a typo!
- Combine with **Pair Extraordinaire** — add a `Co-authored-by` tag to each PR and earn both achievements at once
- Contributing to **open source repos** also counts
- The more you contribute, the faster you stack tiers

---

## ⚠️ Common Mistakes

- Closing a PR without merging — must be **merged** not just closed
- Only opening PRs but not merging them
